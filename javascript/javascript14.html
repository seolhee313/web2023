01. 퀴즈 이펙트 : 정답 확인하기 유형

정답 확인버튼을 누르면 접답이 나오는 효과입니다.

  • 데이터 저장 : 변수
  • querySelector() : 문서에서 특정 CSS 선택자로 요소를 찾는 DOM 메서드입니다.
  • innerText : DOM 요소의 텍스트 내용을 가져오거나 설정하는 데 사용되는 속성입니다.
  • style.속성 = "속성값" : 자바스크립트에 CSS 스타일을 표현하는 방법입니다.
  • classlist : 요소의 클래스를 추가하거나 제거하여 스타일을 설정하는 데 사용됩니다.
  • addEventListener : 자바스크립트에서 이벤트를 처리하는 함수입니다.

02. 퀴즈 이펙트 : 주관식 확인하기 유형

정답을 적으면 맞는지 확인해주는 효과입니다.

  • 데이터 저장 : 변수
  • 데이터 제어하기 : if문
  • trim() : 문자열의 양 끝에 있는 공백을 제거하는 데 사용됩니다.
  • textContent : HTML 요소의 텍스트 내용을 나타내는 속성입니다. 이 속성을 사용하면 JavaScript를 사용하여 HTML 요소의 내용을 수정할 수 있습니다.
  • .value : HTML 폼 요소(input, select, textarea 등)의 현재 값을 가져오거나 설정하는 속성입니다.

03. 퀴즈 이펙트 : 주관식(여러문제) 확인하기 유형

정답을 적으면 맞는지 확인해주는 효과입니다.

  • 데이터 저장 : 배열, 객체
  • 데이터 불러오기 : for문, forEach()
  • 데이터 제어하기 : if문
  • querySelectorAll : 웹 브라우저에서 제공하는 JavaScript 메소드 중 하나입니다. 이 메소드를 사용하면, 주어진 CSS 선택자에 해당하는 모든 요소를 찾아서 NodeList 객체로 반환합니다.
  • innerHTML : HTML 요소의 내용을 가져오거나 설정하는 JavaScript 프로퍼티입니다. 이 프로퍼티를 사용하면 웹 페이지의 특정 요소의 내용을 동적으로 변경할 수 있습니다.
