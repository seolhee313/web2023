• padding : 문자열(String)을 특정한 길이로 맞추기 위해 사용됩니다.
  • padding-bottom : 요소의 아래쪽 내부 여백(padding)을 설정하는 속성입니다.
  • padding-left : 요소의 왼쪽 내부 여백(padding)을 설정하는 속성입니다.
  • padding-right : 요소의 오른쪽 내부 여백(padding)을 설정하는 속성입니다.
  • padding-top : 요소의 위쪽 내부 여백(padding)을 설정하는 속성입니다.
  • page-break-after : 인쇄된 페이지에서 다음 페이지로의 전환을 지정하는 속성입니다.
  • page-break-before : 인쇄된 페이지에서 현재 페이지의 앞에 새로운 페이지를 시작하도록 지정하는 속성입니다.
  • page-break-inside : 인쇄된 페이지에서 요소 내부에서 페이지가 나누어지는 것을 허용하거나 방지하는 속성입니다.
  • perspective : 3차원 공간에서 요소의 원근감(perspective)을 설정하는 속성입니다.
  • perspective-origin : 3차원 공간에서 perspective 속성이 적용된 요소의 원근감(perspective)을 설정하는 중심점(origin)을 지정하는 속성입니다.
  • pointer-events : 요소에 마우스 포인터 이벤트(mouse events)가 적용되는 방식을 지정하는 속성입니다.
  • position : 요소의 위치를 지정하는 속성입니다.
  • quotes : 문자열을 정의할 때 사용하는 기호를 말합니다.
  • resize : JavaScript에서 사용되는 이벤트 중 하나입니다. 이 이벤트는 브라우저 창의 크기가 변경될 때 발생합니다.
  • right : 요소(element)의 오른쪽 경계(edge)가 뷰포트(viewport)의 오른쪽 가장자리에서 얼마나 떨어져 있는지를 설정합니다.
  • tab-size : 탭 문자(tab character)가 표시될 때의 간격을 지정합니다. 기본값은 8이며, length 값을 통해 설정할 수 있습니다.
  • table-layout : 'table' 요소의 레이아웃 방식을 지정합니다.
  • text-align : 텍스트 정렬 방식을 설정합니다.
  • text-align-last : 문단의 마지막 텍스트 정렬 방식을 설정합니다.
  • text-decoration : 속성은 텍스트 라인 속성을 일괄적으로 설정합니다.
  • text-decoration-color : 속성은 글자 라인의 색상 속성을 설정합니다.
  • text-decoration-line : 글자 라인의 위치 속성을 설정합니다.
  • text-decoration-style : 텍스트에 적용되는 선(line)의 스타일을 지정합니다.
  • text-decoration-skip-ink : 언더 라인의 글씨 겹침을 설정합니다.
  • text-decoration-thickness : 글자 라인의 굵기 속성을 설정합니다.
  • text-emphasis-color : 텍스트에 강조 표시 색상을 설정합니다.
  • text-emphasis-position : 텍스트에 강조 표시 위치를 설정합니다.
  • text-emphasis-style : 텍스트에 강조 표시 스타일을 설정합니다.
  • text-emphasis : 텍스트에 강조 표시를 일괄적으로 설정합니다.
  • text-indent : 문단 들여쓰기 속성을 설정합니다.
  • text-justify : 양쪽 정렬을 했을 때 정렬 유형을 설정합니다.
  • text-overflow : 텍스트가 영역을 벗어 났을 때 속성을 설정합니다.
  • text-shadow : 텍스트 그림자를 설정합니다.
  • text-transform : 텍스트의 대소문자 변환 방법을 지정합니다.
  • text-underline-offset : 언더 라인의 오프셋 값을 설정합니다.
  • text-underline-position : 언더 라인의 위치를 설정합니다.
  • top : 요소(element)를 수직 방향(top-down)으로 위치시킬 때 사용됩니다.
  • transform : 요소(element)의 변형(transform)을 지정하는 데 사용됩니다.
  • transform-origin : transform 속성을 사용하여 요소를 변형할 때 변형의 기준점(origin)을 지정하는 데 사용됩니다.
  • transform-style : 3D 변환(transform)이 적용된 자식 요소들의 변환 방법을 지정합니다.
  • transition : HTML 요소의 스타일을 변경할 때 애니메이션 효과를 부드럽게 적용하기 위한 방법 중 하나입니다.
  • transition-delay : transition에서 사용되는 속성 중 하나로, 전환 효과가 시작되기 전에 대기하는 시간을 나타냅니다.
  • transition-duration : CSS transition에서 사용되는 속성 중 하나로, 전환 효과가 일어나는 시간을 나타냅니다.
  • transition-property : CSS transition에서 사용되는 속성 중 하나로, 전환 효과를 적용할 CSS 속성을 지정합니다.
  • transition-timing-function : CSS transition에서 사용되는 속성 중 하나로, 전환 효과의 속도 곡선을 지정합니다.
  • unicode-bidi : 언어의 흐름 방향을 지정하는 데 사용됩니다.
  • user-select : 요소의 텍스트 선택을 가능하게 하거나 막는 데 사용됩니다.
  • vertical-align : 인라인 요소의 수직 정렬을 설정하는 데 사용됩니다.
  • visibility : 요소를 화면에 표시할지 여부를 설정하는 데 사용됩니다.
  • white-space : 줄바꿈 및 공백을 설정합니다.
  • width : 요소의 가로 크기를 설정합니다.
  • word-break : 줄바꿈을 할때 단어를 기준으로 설정하는 속성입니다.
  • word-spacing : 요소 내부의 단어 간 간격을 조절하는 데 사용됩니다.
  • word-wrap : 띄어쓰기가 없는 긴 단어를 어떻게 처리할지 정합니다.
  • z-index : 위치 지정 요소와, 그 자손 또는 하위 플렉스 아이템의 Z축 순서를 지정합니다.
